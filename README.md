# FrameShot

Beautiful device mockups for your app screenshots. 100% client-side, no signup required.

🌐 **Live:** [thakoreh.github.io/frameshot](https://thakoreh.github.io/frameshot/)

## Features

- **7 Device Frames:** iPhone 15, iPhone 15 Pro, Pixel 8, Galaxy S24, iPad Pro, MacBook Pro 16", iMac 24"
- **12 Gradient Presets** + custom two-color gradients with angle control
- **Export-Ready Sizes:** App Store, Play Store, Twitter/X, ProductHunt, OG images
- **Title & Subtitle Overlay** for landing page mockups
- **100% Client-Side** — your screenshots never leave your browser
- **Dark & Light Mode** support
- **No Signup Required**

## Pricing

- **Free:** 5 PNG exports, all devices, all gradients
- **Pro ($15 one-time):** Unlimited exports, all features, lifetime updates

## Tech

- Next.js 16 (App Router) + Tailwind CSS + TypeScript
- SVG device frames rendered in React
- html-to-image for high-res PNG export
- Deployed on GitHub Pages
